* Digital Voice Assistant:
This application is a virtual assistant that can accept voice input, translate it into text using speech-to--text technology, forward the text to OpenAI's GPT-3 model, get a reply, translate the text back into speech using text-to--speech technology and subsequently play it back to the user.

In this project I used an Open Source LLM model from Hugingface
