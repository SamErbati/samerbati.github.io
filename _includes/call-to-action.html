Projects

    AI Projects

    • RAG-Chatbot Development: Developed a Retrieval-Augmented Generation (RAG) chatbot capable of providing more accurate and contextually relevant responses by combining retrieval-based and generative models.
    • Image Captioning with Generative AI: Implemented a generative AI model that can automatically generate descriptive captions for images, enhancing accessibility and user experience.
    • ChatGPT-like Website: Created a website similar to ChatGPT, enabling users to interact with an advanced conversational AI for various applications, including customer service and personal assistance.
    • Voice Assistant Creation: Developed a voice assistant capable of understanding and executing voice commands, providing a hands-free user experience for various tasks.
    • Generative AI-powered Meeting Assistant: Designed an AI assistant to support meetings by generating summaries, action points, and contextual information, improving meeting efficiency and follow-up.
    • Summarize Private Data with Generative AI: Built a generative AI system to analyze and summarize private data, offering users concise and relevant insights from large datasets.
    • Voice-enabled AI Translator Assistant: Created an AI assistant that provides real-time voice translation, facilitating communication across different languages and enhancing user accessibility.

    Optimization Algorithms Projects

    • ILP Optimization Models: Developed Integer Linear Programming (ILP) optimization models to improve latency, reliability, and bandwidth consumption in NFV-enabled networks, enhancing network performance and efficiency.
    • Heuristic Optimization Algorithms: Designed heuristic algorithms that outperform existing solutions in terms of time and space complexity, providing more efficient approaches to complex optimization problems.

    IoT Projects

    • Sigfox Gateway and End Nodes: Implemented Sigfox gateways and end nodes for diverse IoT applications, enabling low-power, wide-area network communication for IoT devices.
    • NB-IoT Gateway and End Nodes: Created NB-IoT gateways and end nodes, facilitating narrowband IoT connectivity for enhanced device communication in various IoT scenarios.
